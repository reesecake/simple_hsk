Introduction

Welcome to the Simple HSK docs! Here you can find info on how Simple HSK is written and what is happening in the background, from the Spring Boot REST API to the React components made to provide users with a platform to practice their HSK vocabulary skills.

The Simple HSK GitHub repository is not currently public and cannot be cloned/forked nor can the source code be viewed.

GitHub

Simple HSK is available on GitHub and connected to CircleCI and Heroku

See GitHub Repo

Requirements

  • JDK 11.0 or higher

Install

Get a copy of the code

See more on how to do this

$ git clone git@github.com:YOUR_USERNAME/simple_hsk.git

Install node and npm with the frontend-maven-plugin

$ mvn install

or separately

$ mvn frontend:install-node-and-npm
$ mvn frontend:npm

Run

Start the Spring Boot application

$ ./mvnw spring-boot:run

Run webpack to pack the React components

$ mvn frontend:webpack
